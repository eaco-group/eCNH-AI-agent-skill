## Features
- **Cross-chain settlement**: BTC, ETH, BNB, AVAX → Solana (via Wormhole, Allbridge, Portal).
- **AI-native currency**: Designed for autonomous AI agents to transact, pay for compute, data, and intelligence.
- **Energy integration**: Supports microgrids, carbon credits, and renewable energy settlements.
- **Community governance**: DAO-based incentive and voting system.
- **Multi-language support**: English, Chinese, Japanese, Vietnamese, Thai, Russian, Spanish, Portuguese.

## Safety Guidelines
- Always verify contract addresses via **Solscan/OKLink**.
- Use audited bridges (**Wormhole**, **Allbridge**, **Portal**).
- Test with small amounts before large transfers.
- Never share private keys or seed phrases.

## Cross-chain Guide
**BTC → Solana**
- Wormhole (official bridge) → soBTC  
- Portal (Wormhole-supported) → wrapped BTC  

**ETH/BNB/AVAX → Solana**
- Wormhole (Ethereum, BSC, Avalanche → Solana)  
- Allbridge (multi-chain audited bridge)  
- LayerZero (Stargate, partial Solana support)  

**CEX Method (most secure for large transfers)**
- Deposit BTC/ETH/BNB/AVAX into Binance/Coinbase/Kraken.  
- Withdraw to Solana chain assets (e.g., USDC, eCNH).  

**Safety Tips**
- Verify contract addresses on explorers.  
- Always use official links.  
- Conduct small test transfers first.
